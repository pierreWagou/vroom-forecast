# Exploration

Independent uv project for exploratory data analysis via Jupyter notebooks.
The notebook is Jupytext-synced (`.py` percent format).

## Running

```bash
uv run --project exploration jupyter notebook --no-browser --port=8888
```

## Key files

- `exploration.py` — Jupytext-synced notebook (percent format)
- `pyproject.toml` — Dependencies: training deps + matplotlib, seaborn, jupyter
